# Spring Boot Reactive REST API 
Proof of concept for Spring Boot Reactive REST API with embedded H2 database


#### Technologies
1. Spring Boot 2.6.0
2. Spring Data R2DBC - Reative database drivers
3. Spring Webflux with Project Reactor
4. Lombok
5. Maven
6. H2 database
